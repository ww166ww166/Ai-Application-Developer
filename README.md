# Ai-Application-Developer
这是一个关于 AI 应用开发的学习记录仓库。

## 目录结构
* `/day001/`: 第一天的基础练习，包含：
    * `hello.py`: 环境测试
    * `intro.py`: 项目简介
    * `practice1.py`: 基础练习
    * `why.py`: 为什么学习 AI 开发

## 如何运行
确保你安装了 Python 3.x，然后在终端运行：
python day001/hello.py


.md 代表 Markdown 格式，它支持简单的排版（如 # 表示大标题，* 表示列表）。
